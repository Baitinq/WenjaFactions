# WenjaFactions (fork of iHCF)

Dependencies
- PaperSpigot 1.7 (use one provided, put API first)
- Commons Lang 3
- Commons Collections 4.1
- GNU Trove 3.0.3
- Guava 21.0
- Lombok (plugin + library)
- WorldEdit 6.1.5
- ProtocolLib 3.7.0

Trove, Guava, and both Commons libraries need to be exported with the project.
